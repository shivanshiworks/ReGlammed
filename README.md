# ReGlammed 👗

A modern iOS marketplace built with **SwiftUI** and **Firebase** that allows users to buy, sell and rent pre-loved fashion.

Designed with a clean luxury aesthetic using a custom cream, blue, yellow and brown color palette, ReGlammed focuses on giving fashion a second life while making sustainable shopping simple and enjoyable.

## Features

### Authentication

* Google Sign In
* Firebase Authentication
* User Profiles
* Phone Verification (architecture ready)

### Home

* Featured Listings
* Browse Buy
* Browse Rent
* Search
* Recent Searches
* Featured Carousel

### Buy & Rent

* Real-time Firebase listings
* Search
* Filter by Category
* Filter by Brand
* Filter by Size
* Sort by:

  * Recent
  * Oldest
  * Price Low → High
  * Price High → Low

### Listing Details

* Multiple Images
* Seller Information
* WhatsApp Contact
* Share Listing
* Save Listing

### Upload Listings

* Upload Images
* Sell or Rent
* Category Selection
* Size Selection
* Condition Selection
* Pricing
* Rental Duration
* Validation
* Loading State

### Profile

* User Profile
* My Listings
* Saved Listings
* Logout
* WhatsApp Verification

## Getting Started

Clone the repository:

```bash
git clone https://github.com/shivanshiworks/ReGlammed.git
```

Open the project in Xcode.

Configure your own Firebase project by adding your own `GoogleService-Info.plist`.

Run on an iOS Simulator or physical device.
